Research

My research has focused on data mining in education, and data-driven methods to improve student learning. I use statistical models and machine learning algorithms to determine what needs or topics have to be addressed next with a student. The data assist teachers to plan for appropriate pedagogy and targeted learning to more effectively scaffold the learning needs of their students. Here are a brief summary of my research:

  • Mining student response data for knowledge using Q-matrix method
  • Using Expectation-Maximization(EM) algorithm for online calibration in cognitive diagnostic latent class model
  • A non-parametric hypothesis testing to detect parameter shift in computerized adaptive testing
  • A stepwise method to detect parameter shift in test items

Dissertation

For my Ph.D. dissertation, I explored optimal design to estimate high-dimensional item response model parameters in computer-based online testing framework, which has the following parts:

  • Construct a high-dimensional logistic regression model and apply it to computerized adaptive testing
  • Derive Fisher information matrix of estimating model parameters
  • Explore optimal design to maximize determinant value of fisher information matrix
  • Develop Expectation-Maximization algorithm to estimate model when both latent variables and model parameters are unkonwn
